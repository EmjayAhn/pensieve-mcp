대시보드 사용 가이드 MCP 설정

🔧 MCP 서버 설정

Claude Desktop 앱과 Pensieve를 연결하는 방법입니다

1

Claude Desktop 앱 설치

먼저 Claude Desktop 앱이 설치되어 있어야 합니다. 아직 설치하지 않았다면 여기서 다운로드하세요.

✓ macOS, Windows, Linux 지원

✓ 최신 버전 권장

2

Pensieve MCP 프로젝트 클론

먼저 GitHub에서 Pensieve MCP 프로젝트를 클론받으세요:

git clone https://github.com/EmjayAhn/pensieve-mcp.git
cd pensieve-mcp

프로젝트를 클론받은 경로를 기억해두세요. 다음 단계에서 사용됩니다.

3

MCP 서버 설정

Claude Desktop 앱의 설정 파일을 열어 MCP 서버 정보를 추가합니다:

설정 파일 위치:

macOS ~/Library/Application Support/Claude/claude_desktop_config.json
Windows %APPDATA%\Claude\claude_desktop_config.json
Linux ~/.config/Claude/claude_desktop_config.json

또는 Claude Desktop 앱에서 직접 열 수 있습니다:

Settings → Developer → Edit Config

다음 설정을 추가하세요:

{
  "mcpServers": {
    "pensieve": {
      "command": "/opt/homebrew/bin/python3.12",
      "args": ["/Users/username/pensieve-mcp/mcp_server/server_api.py"],
      "env": {
        "PENSIEVE_API_URL": "https://pensieve-api.whitesea-c963c99f.koreacentral.azurecontainerapps.io"
      }
    }
  }
}

주의사항:

  • command: Python 실행 파일 경로를 시스템에 맞게 수정하세요
    • macOS (Homebrew): /opt/homebrew/bin/python3.12
    • macOS (기본): /usr/bin/python3
    • Windows: python 또는 C:\\Python312\\python.exe
  • args: 클론받은 pensieve-mcp 프로젝트 경로 + /mcp_server/server_api.py
  • PENSIEVE_API_URL: 클라우드 API 주소 (변경하지 마세요)
4

Claude 앱 재시작

설정을 저장한 후 Claude Desktop 앱을 완전히 종료하고 다시 실행하세요.

재시작 방법:

  1. Claude Desktop 앱 완전 종료 (Cmd/Ctrl + Q)
  2. 앱 다시 실행
  3. 새 대화 시작
5

연결 확인

Claude와의 새 대화를 시작하고, MCP 서버가 연결되었는지 확인하세요:

# Claude에서 다음과 같이 물어보세요:
"Pensieve에 저장된 대화를 불러와줘"

MCP 서버가 정상적으로 연결되었다면 저장된 대화 목록이 표시됩니다!

❓ 문제 해결

연결이 안 될 때

  • Python 경로가 올바른지 확인 (터미널에서 which python3 실행)
  • server_api.py 파일 경로가 정확한지 확인
  • 설정 파일의 JSON 형식이 올바른지 확인
  • Claude 앱을 완전히 재시작했는지 확인

Python 경로 찾기

  • macOS/Linux: 터미널에서 which python3 또는 which python3.12
  • Windows: 명령 프롬프트에서 where python
