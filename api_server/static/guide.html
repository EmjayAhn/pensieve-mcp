Pensieve 사용 가이드

AI 대화를 효율적으로 관리하는 방법을 알아보세요

✨ 주요 기능

자동 저장

모든 AI 대화가 자동으로 저장되어 언제든 다시 확인할 수 있습니다.

강력한 검색

키워드로 과거 대화를 빠르게 검색하고 찾을 수 있습니다.

태그 관리

대화를 태그로 분류하여 체계적으로 관리하세요.

공유 기능

중요한 대화를 팀원과 쉽게 공유할 수 있습니다.

📚 사용 방법

1. MCP 서버 설정

Claude Desktop 앱에서 MCP 설정을 추가합니다:

  • 설정 메뉴에서 Developer → Edit Config 선택
  • 제공된 설정 코드를 복사하여 붙여넣기
  • Claude 앱 재시작

2. 대화 저장하기

Claude와 대화할 때 다음과 같이 말하세요:

"Pensieve에 이 대화를 저장해줘" "이 대화를 'Python 학습'이라는 제목으로 저장해줘" "'React 컴포넌트 설계'로 저장하고 태그는 'react', 'frontend'로 해줘"

💡 자연스럽게 말하면 Claude가 적절한 제목과 태그를 제안해줍니다!

3. 대화 검색하기

저장된 대화를 찾고 싶을 때:

"Pensieve에서 Python 관련 대화를 찾아줘" "저장된 대화 중에서 '리액트'가 포함된 것들 보여줘" "지난주에 저장한 대화들을 불러와줘"

웹 대시보드에서도 검색 기능을 사용할 수 있습니다.

4. 태그로 찾기

특정 주제의 대화만 보고 싶을 때:

"javascript 태그가 달린 대화들 보여줘" "프로그래밍 관련 대화만 골라서 보여줘"

태그를 활용하면 주제별로 체계적으로 대화를 정리할 수 있습니다.

💡 활용 팁

의미있는 제목 사용

나중에 쉽게 찾을 수 있도록 대화 내용을 잘 설명하는 제목을 사용하세요.

태그로 체계적 정리

프로젝트, 주제, 날짜 등 다양한 태그를 활용해 대화를 분류하세요.

정기적인 정리

주기적으로 오래된 대화를 정리하고 중요한 내용은 별도로 백업하세요.

팀 협업 활용

팀원과 지식을 공유하고 함께 문제를 해결하는 데 활용하세요.
